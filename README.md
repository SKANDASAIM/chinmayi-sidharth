# 💖 Wedding Invitation Website – Chinmayi & Siddharth

A modern, elegant, and fully responsive digital wedding invitation website designed for **Chinmayi M & Siddharth V. Hallera**. The website features a premium glassmorphism-inspired UI with smooth animations and is optimized for desktop, tablet, and mobile devices.

## ✨ Features

* 🎉 Beautiful and responsive wedding invitation design
* 🖼️ Custom couple image section
* 💍 Reception and Muhurtham event details
* ⏳ Live countdown timer to the wedding
* 📍 Direct Google Maps venue link
* 📅 Add-to-calendar button
* 💌 "Leave Your Wishes" section integrated with **Formspree** for email submissions
* 📱 Mobile-friendly and cross-browser compatible
* 🎨 Elegant typography and modern UI with soft gradients and animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Formspree (for contact/wishes form)

## 🚀 Getting Started

1. Clone the repository.
2. Replace the couple image with your own in the `images` folder.
3. Update the Google Maps venue link if required.
4. Configure your Formspree endpoint for the wishes form.
5. Open `index.html` in a browser or deploy using GitHub Pages.

## ❤️ Live Experience

Designed to provide guests with an interactive and memorable digital invitation experience while allowing them to send their heartfelt wishes directly to the couple via email.
